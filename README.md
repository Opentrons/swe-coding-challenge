## Project Setup

Install dependencies:

```shell
yarn
```

Run the app in the development mode on [http://localhost:8080]:

```shell
yarn start
```

Run tests

```shell
# run tests
yarn test

# run tests in watch mode
yarn test:watch
```

Run formatting and linting

```shell
# format JS, JSON, HTML, and Markdown
yarn format

# lint JS and CSS
yarn lint
```
